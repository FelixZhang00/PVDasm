Borland C++ 5.02 Help update 9/24/97
=====================================

This Help update fixes problems associated with generating a full text 
search database using the WinHelp Find tab. The update also fixes the 
bitmap display problems in the Win32 Reference.


The patch file, BC52HLP1.ZIP, contains the following files:

95GUIDE.CNT	Contents file for 95GUIDE.HLP.
SETUPAPI.CNT	Contents file for SETUPAPI.HLP.
SOCK2.CNT	Contents file for SOCK2.HLP.
WIN32.HLP	Win32 Reference.
README.TXT	This file.


Where to copy the files
-----------------------

Copy the 95GUIDE.CNT, SETUPAPI.CNT, SOCK2.CNT, and WIN32.HLP files to 
the BC5\HELP directory, overwriting the existing files of 
the same names.

IMPORTANT
---------

Before using the updated Help files, remove any hidden .GID (general 
index) files and .FTS (full text search) files in the BC5\Help directory.
